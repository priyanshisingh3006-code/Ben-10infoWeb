# Ben 10 Fan Website

## What this project is

This is a simple multi-page frontend website based on the Ben 10 cartoon. The site includes different pages for main characters like Ben, Gwen, Kevin, Grandpa Max, and Professor Paradox, along with separate pages for some of my favorite aliens like Alien X and Rath.

The website also includes a dedicated page for Charmcaster, who I find to be one of the most interesting female characters in the series.

The goal was to create something that feels like a small fan-made information site rather than a perfect or overly polished project.

---

## Why I made this

I made this project mainly for practice. I am still new to coding and wanted to try building a proper multi-page website instead of just small basic projects.

Also, I really like Ben 10. It is one of those cartoons that had a strong story, interesting characters, and concepts like time travel, multiverse, and different timelines. So I thought it would be fun to build something around it.

---

## How I made this

I built this using basic HTML, CSS, and a bit of JavaScript.

* HTML was used to create all the pages and structure
* CSS was used for styling and making a dark theme inspired by Ben 10
* JavaScript was used for things like the hamburger menu and small interactions

I tried to keep the structure simple and not overcomplicate things since I am still learning.

I also used a little bit of AI help mainly to fix JavaScript errors and understand what was going wrong when something didn’t work.

---

## What I struggled with

The hardest part for me was JavaScript. I am not very comfortable with it yet, so things like the menu toggle and active link system took time to understand and fix.

I also struggled with layout issues in CSS. Sometimes things looked fine on one screen but broke on another, so I had to adjust spacing and structure multiple times.

Another challenge was managing multiple pages and keeping everything consistent across them.

---

## What I learned

From this project, I learned:

* How to structure a multi-page website
* Better understanding of HTML layout and linking pages
* Basic JavaScript usage for interactivity
* How CSS affects layout, spacing, and responsiveness
* How to debug issues when things don’t work as expected

I also learned that even a simple project can take time when you are doing everything on your own.
